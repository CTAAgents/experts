# 中国生猪市场整合研究

- **日期**: 2026-07-07
- **来源**: 杨志波, 2017年12月出版, 微信读书
- **分类**: 论文研报
- **标签**: 生猪, 市场整合, 猪周期, 区域经济, 产业组织

## 核心要点

- 核心议题:我国生猪市场存在显著的区域分割,行政壁垒导致价格信号失真

- 影响因素:交通物流(冷链)vs行政干预(地方保护)vs自然条件(疫病)的三重作用

- 政策建议:建设统一大市场,减少地方补贴扭曲,推动规模化养殖整合

- 数据局限:书中数据分析至2015年,但理论框架(市场分割-价格传导-整合路径)仍适用
