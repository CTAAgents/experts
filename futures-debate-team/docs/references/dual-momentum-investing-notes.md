# 双动量投资：高回报低风险策略—加里·安东纳奇

- **日期**: 2026-07-09
- **来源**: 微信读书 | 加里·安东纳奇 著 动量投资经典必读
- **分类**: 分析方法
- **标签**: 双动量, 绝对动量, 相对动量, 安东纳奇, GEM策略, 全球股票动量, 资产配置, 12个月动量

## 核心要点

- 双动量=绝对动量(市场vs无风险)+相对动量(美股vs全球)

- 核心策略:三资产(标普500/MSCI ACWI ex US/综合债券),每月比12个月收益率

- 回测(1974-2013):年化17.43% vs 标普500~11.4%,最大回撤22.7%

- 2008年金融危机中通过切换至债券实现正收益

- 最大优势:简单透明,年均交易仅1.35次

- 12个月窗口:过滤短期噪音+降低交易频率+控制税收成本

- 与中国蛋卷斗牛二八轮动的区别:①前者12月动量vs后者20天动量②三资产相关性极低vs沪深300与中证500高相关

- 作者背景:哈佛MBA,行为金融与动量投资研究先驱

## 可应用场景

- CTA/股票趋势跟踪系统中的宏观过滤层

- 多资产动量轮动策略

- 商品期货版双动量(商品指数+债券)

## 详细分析

### 双动量的两层结构

```
绝对动量（时机判断）        相对动量（资产选择）
   S&P 500 > T-bill     VS    S&P 500 vs MSCI ACWI ex US
   ───────────────            ─────────────────────────
   是 → 进入股票             选收益更高的那个
   否 → 持有债券             同时持有或全部债券
```

**双动量 = 先判断"该不该入场"（绝对动量），再判断"入哪个市场"（相对动量）。**

### 核心策略（GEM = Global Equity Momentum）

每月底执行的操作：
```
1. 计算S&P 500过去12个月的总收益率
2. 若 > 美国短期国债利率 → 绝对动量满足，进入第3步
3. 比较标普500 vs MSCI ACWI ex US的12个月收益率
4. 持有收益率更高的那个（不交叉持有）
5. 若绝对动量不满足 → 全部持有综合债券指数
6. 等待下个月底，重复
```

### 与中国"蛋卷斗牛二八轮动"的对比

这是一个重要的对照实验——同样的双动量思想，不同的实现参数，结果截然不同：

| 维度 | 双动量（安东纳奇） | 蛋卷斗牛二八轮动 |
|------|------------------|----------------|
| 动量窗口 | 12个月 | 20天 |
| 资产相关性 | 标普500 vs MSCI全球(除美) 相关性低 | 沪深300 vs 中证500 相关性高 |
| 交易频率 | 年均1.35次 | 远高于前者 |
| 税收摩擦 | 低 | 高 |
| 实际结果 | ✅ 年化17.43% | ❌ 追涨杀跌损耗严重 |

### 对读者系统的启发

**1. 为DC20/DC55系统增加宏观过滤层**
- 当前你的趋势跟踪系统直接对单个品种做多空判断
- 可以借鉴双动量的"绝对动量"思想：**当农产品指数/有色指数/黑色指数的12个月动量为正时，才允许其下的品种开多头仓位**
- 这相当于给品种级的趋势跟踪加了一层板块级的趋势确认

**2. 商品期货版双动量**
- 你的系统已经覆盖14条产业链，可以设计一个"产业链双动量"
- **绝对动量**: 某商品指数（如南华商品指数）> 国债利率？（判断是否应该做多商品）
- **相对动量**: 在强势产业链（黑色/有色/能化/农产品）中选择12个月动量最强的

**3. 与FDT辩论系统的集成**
- 双动量的决策逻辑可以集成到闫判官的裁决规则中
- 例如：如果商品指数绝对动量为负 → 闫判官在裁决时自动对多头方向论据施加额外权重 penalty

**4. 对因子挖掘Agent的启示**
- 双动量是"最简单的有效因子"——只用过去12个月收益率这一个信号就能实现年化17.43%
- 这说明因子挖掘Agent中生成的复杂因子，**首先必须与简单因子（如12个月动量）做边际贡献对比**——如果一个因子的信息比12个月动量还少，它的价值就存疑
