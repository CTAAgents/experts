## 核心要点

- 国内期货信息源可按"权威层级+聚合度"分为三层：第一层监管/交易所（权威源头，免费基准），第二层综合资讯聚合站（行情+快讯+研报+社区一体化），第三层产业垂直聚合（按品种锚定现货+基差+库存+开工）

- 第一层核心8站：证监会、中期协、上期所、大商所、郑商所、中金所、广期所、上期能源；连带宏观数据源（统计局/海关/发改委/央行）

- 第二层5站：期货日报网（正统研报）、金十数据（宏观快讯闹钟）、东方财富期货（行情+社区）、新浪财经期货（内外盘均衡）、和讯期货（老牌门户+学院）

- 第三层按品种分：黑色系（我的钢铁网/找钢网/煤炭资源网）、有色系（上海有色网/长江有色/上海金属网）、能化系（隆众/百川/卓创/金联创）、农产品（我的农产品网/USDA/中糖协）

- 通用基差工具：生意社100ppi.com（500+商品基准价免费看基差）

- 机构级付费终端（个人站不可替代）：Wind/同花顺iFinD/新华财经终端/卓创红期 — 宏观+行业+期现+仓单+研报全打通支持导出建模

- 组合用法：金十盯宏观数据 → 期货日报看行业研报 → 东财/新浪看社区情绪 → 垂直站锚定品种基差，基本覆盖期货信息需求

## 可应用场景

- 专家团定性取证：futures-debate-team memory/info_portals.md 即本三层级框架的结构化落地（另含品种映射速查表），明鉴秋补充定性证据时优先查

- 自主研究索引：研究员查数据/研报时按三层级框架定位权威源，避免信息孤岛

- 信息源审计：新接入数据源前，先对照此三层级框架确认其在信息生态中的定位与权威性

- 交易辅助：按品种速查推荐门户（如PTA→郑商所+隆众+生意社），快速定位对应供需/基差数据

- 铜铝锌 → 上期所+上海有色网+长江有色
- 豆粕/玉米/生猪 → 大商所+我的农产品网+USDA
- 黄金/白银 → 上期所+CME中文
- 棉花/白糖 → 郑商所+中糖协+生意社
