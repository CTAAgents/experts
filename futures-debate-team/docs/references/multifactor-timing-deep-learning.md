# Multifactor Timing with Deep Learning

- **日期**: 2026-07-07
- **来源**: Cotturo, Liu, Proner (2026), Journal of Financial Econometrics
- **分类**: 论文研报
- **标签**: 深度学习, 多因子定时, LSTM, DMT, 因子择时

## 核心要点

- DMT架构:双LSTM分别建模宏观/金融状态+共享层,Shapre 0.82

- 多任务学习(MT)强制共因结构,天然正则化比单因子模型好

- 最强预测变量:失业率>杠杆类>盈利类>货币类

- 非线性交互贡献大,线性模型miss

- 14bp交易成本吃掉约一半alpha

## 可应用场景

- 链证源宏观状态标签

- 辩论系统架构验证

- 风控明交易成本核算
