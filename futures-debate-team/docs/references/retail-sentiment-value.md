# 论坛散户观点有价值吗？——散户舆情选股投资手册

- **日期**: 2026-07-07
- **来源**: 国金证券《Alpha掘金系列之二十三》/ 量化智投(2026-06-23报告,2026-07-07推送) https://mp.weixin.qq.com/s/g7XocBsvCGRqIFaj-8Sokw
- **分类**: 论文研报
- **标签**: 散户舆情, 股吧, 选股, FinBERT2, BERTopic, 另类数据, 情感分类, 主题分类, 噪音交易者, 小模型

## 核心要点

- FinBERT2垂类情感分类F1=92.95% 高于 Claude-3.5-Sonnet 83.10%，独立验证专用小模型优于通用大模型

- 双支柱框架：情感分类(FinBERT2)加主题分类(BERTopic+LLM归纳)

- 市值越小舆情因子越有效：中证1000(IR1.78)大于中证500(1.53)大于沪深300(1.12)

- 情感与技术及情绪占比IC为负(反转效应)，基本面占比IC为正

- 观点多空划分不显著，讨论主题才有增量信息

- 噪音交易者主导度(技术或情绪占比)在热门股中越高越差，可作见顶风险信号

## 可应用场景

- CTA路径B研报过滤可借鉴主题占比思路;辩论专家团信号评估应看讨论维度而非仅多空方向;噪音交易者主导度可移植为期货品种趋势见顶风险监控
