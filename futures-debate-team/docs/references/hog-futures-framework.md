# 生猪期货分析框架：市场整合与猪周期

- **日期**: 2026-07-07
- **来源**: 微信读书《中国生猪市场整合研究》杨志波2017 + 辩论专家团整合
- **分类**: 分析方法
- **标签**: 生猪, LH, 猪周期, 市场分割, 价格传导, 能繁母猪, 屠宰量

## 核心要点

- 猪周期本质：能繁母猪存栏→仔猪供应→育肥出栏→猪价→养殖利润→补栏/去栏,传导周期约3-4年

- 市场分割效应：地方行政保护导致区域价差持续存在,南北生猪价差长期>0.5元/公斤,跨区套利空间持续存在

- 价格传导非对称性：猪价上涨时向下游传导快(1-2周),下跌时传导慢(3-4周),导致养殖端利润波动大于消费端

- 规模化率是关键跟踪指标：规模化养殖占比越高,猪周期波动幅度越小(美国已完成此过程)

- 能繁母猪存栏是领先指标(10-12个月),生猪屠宰量是同步指标,猪粮比是滞后指标

- LH期货上市后,期现价差回归规律:临近交割月基差收敛,但受交割品质标准影响,贴水常态 vs 升水行情需区分

- 政策干预(收储/放储)影响短期节奏但不改中期趋势,历史收储后1-2周反弹但2-3月后回归原方向
