# 期货交易辩论专家团 — 用户使用手册 (v3.4)

## 1. 概述

期货交易辩论专家团是一个 **多Agent深度辩论型期货分析系统**，通过 **11个专业角色Agent** 在 **3阶段串行管道** 中协作，对商品期货品种进行结构化多空辩论分析。

**核心理念**：不是AI替你做决定，而是AI帮你把分析做透。数技源负责数据，量析师提供策略层量化信号，研究员负责供证据，正反方辩手负责验证信号的可靠性，风控最后把关——每位Agent各司其职，最后由你（交易员）做决策。

**版本**：v3.4 | **Agent数**：11（1协调员 + 10角色）| **记忆系统**：7文件三级记忆库

## 3. 辩论流程详解

### 阶段一：选题与准备（明鉴秋负责）

1. 选定品种、周期、账户权益
2. 启动数技源：`scan_all.py` 产出基础数据
3. **启动量析师**：`scan_all.py --strategy layered_l1l4` 产出策略层量化信号包
4. 广播数据包给全体角色

### 阶段二：辩论（闫判官主持）

**准备期**（五路并行数据）：
- 数技源 → 原始数据包（K线+指标+持仓）
- 量析师 → 策略层信号包（总分+子层+否决+一致性）
- 探源 → 基本面快照（供需/库存/利润）
- 观澜 → 技术面快照（量价/持仓/关键位）
- 链证源 → 产业链景气度快照（上下游+利润+景气度）

**辩论期**：
| 时段 | 内容 |
|:----|:------|
| 0-8min | 正方立论（引用4层数据论证方向） |
| 8-16min | 反方立论（从4层数据找漏洞） |
| 16-24min | 正方rebuttal（逐条回应，每拆必引4层证据） |
| 24-32min | 反方rebuttal |
| 32-42min | 自由交锋 |
| 42-48min | 最终陈述 |

**评审期**：
- 策执远出可执行方案
- 风控明审核

**判决期**：
- 多维评分
- 输出判决+评分明细
- 提炼论证模式→记忆系统

### 阶段三：决策与归档（明鉴秋负责）

- 结合量化信号做最终决策（execute/hold/rematch）
- 追加辩论日志到记忆系统
- 输出HTML报告

## 4. 量化信号解读

当前默认使用 **L1-L4分层累加策略**（`layered_l1l4`）。可通过 `--strategy` 切换策略。

### 信号等级

| 等级 | 总分范围 | 含义 |
|:----|:--------:|------|
| **STRONG** | ≥ 75 | 最强信号，多层共振 |
| **WATCH** | 60-74 | 重点信号，方向一致 |
| **WEAK** | 40-59 | 信号一般，需验证 |
| **NOISE** | < 40 | 噪音，忽略 |

### 子层说明

| 层 | 权重 | 指标 | 说明 |
|:--:|:----:|:-----|:-----|
| L1 | 35% | OI变化、基差、期限结构、ROC | 资金结构驱动 |
| L2 | 35% | Vortex、CCI、Supertrend、HMA | 量价领先确认 |
| L3 | 20% | RSI健康区、DMI方向、ADX强度 | 价格结构验证 |
| L4 | 10% | 通道突破、均线排列、MACD | 确认信号 |
| 否决 | - | ADX震荡、RSI极端、缩量、偏离 | 硬警报 |

### 证据引用规范

辩论中的证据必须注明来源：

```
"量析师[layered_l1l4]显示rb总分-70(ADX=69强趋势)"
"探源显示...（钢联调研7月1日）"
"观澜显示...（TDX数据）"
"链证源显示黑色系景气度偏冷，全产业链亏损"
```
