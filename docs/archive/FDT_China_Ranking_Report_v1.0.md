## 1. 报告摘要

### 核心结论

| 赛道 | 排名 | 说明 |
|:-----|:----:|:-----|
| **中国期货 CTA 多Agent LLM 系统** | **🥇 第 1 名** | 全网唯一专注62品种的生产级系统 |
| **多Agent 交易决策系统 (全品类)** | **🥇 工程深度第 1 / 🥈 综合第 2** | 自进化/鲁棒性/测试/文档/实盘领先; 社区影响力低于 TradingAgents |
| **学术影响力** | — | 空白（TradingAgents/ProFinAgents/FinSight 有论文或高校背景） |

### 关键数据点

- **中国期货CTA赛道**: 全网无竞品，FDT 是唯一系统
- **AI工程深度**: 自进化/5层鲁棒性/339测试/12份Harness文档 — 超越 TradingAgents
- **最大竞争对手**: TradingAgents (清华系, 66.5K stars)
- **最大短板**: 无社区影响力、无学术论文、无公开业绩

## 3. 按赛道排名分析

### 3.1 赛道一：中国期货 CTA 多Agent LLM 系统

**🥇 第 1 名 — 全网唯一**

| 排名 | 系统 | 品种 | 聚焦 | 生产运行 | 说明 |
|:----:|:-----|:----:|:-----|:--------:|:-----|
| **1** | **FDT** | **62个品种** | **中国期货 CTA** | **是** | **全网唯一** |
| — | 无竞品 | — | — | — | 其他系统全部聚焦股票 |

**关键依据**：

- TradingAgents → A 股/美股
- AI Berkshire → 港/美/A 股价值投资
- Vibe-Trading → 美股量化研究
- FinSight → A 股研究
- ProFinAgents → 通用金融
- 起手Ai Quant Agent → 量化通用（模拟运行）

**没有一个系统专门针对中国期货 CTA 市场**。FDT 是唯一涵盖 62 个商品+金融期货品种、支持趋势跟踪/均值回归/套利/配对/价差/基差/宏观轮动/多因子 8 大策略管线的多Agent LLM 系统。

## 4. FDT vs TradingAgents 深度对比

TradingAgents 是中国市场最直接的可比系统，以下是逐项对比：

### 4.1 基础信息

| 维度 | **FDT** | **TradingAgents** |
|:-----|:-------|:------------------|
| 市场聚焦 | **中国期货(62品种)** | A股/美股 |
| 策略类型 | CTA (趋势/反转/套利/价差/基差/宏观) | 股票交易 |
| 团队 | 个人开发者 | **清华系团队** (Tauric Research) |
| GitHub Stars | 0 (私有) | **66.5K** |
| 学术论文 | 无 | **arXiv + 方正金工研报** |
| 开源协议 | 私有 | 开源 (MIT) |

### 4.2 Agent 架构

| 维度 | **FDT** | **TradingAgents** |
|:-----|:-------|:------------------|
| Agent 数量 | **9** | 7 |
| 角色分工 | **9个独立角色, 边界钉死(含"不做什么")** | 分析师/研究员/交易员/风控/基金经理 |
| 辩论结构 | **P4并行辩论 + P5串行裁决 + 条件分歧路由** | 多轮辩论 + 协调者 |
| 数据源 | **P3按需并行三源(链证源/观澜/探源)** | 分析师并行 |
| 调度架构 | 明鉴秋统一调度, 闫判官有调度决策权 | 基金经理调度 |

### 4.3 工程能力

| 维度 | **FDT** | **TradingAgents** |
|:-----|:-------|:------------------|
| 自进化 | **✅ 5步全链路 (生产运行)** | ❌ 无 |
| 鲁棒性 | **✅ 5层防线 + D06降级表** | ❌ 无系统设计 |
| 测试覆盖 | **✅ 339+ 测试, 13+ 文件** | ❌ 无公开测试 |
| 文档规范 | **✅ 12份Harness文档 + 流程文档** | README + 论文 |
| 存储架构 | **✅ PostgreSQL OLTP+OLAP 混合** | 文件存储 |
| 实盘连接 | **✅ CTP信号输出 (直接交易指令)** | 分析报告, 无执行 |
| 多模型支持 | ❌ 单模型 (DeepSeek) | **✅ o1 + GPT-4o-mini 混合** |

### 4.4 核心差异总结

