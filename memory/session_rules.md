# 子周期K线会话划分规则

> 确立日期: 2026-07-10 | 来源: 掌柜提供交易所K线周期规范文档
> **R0规则**: 时间周期切片以通达信/文华/博易为准，不一致则转换
> **R26规则**: 所有OHLCV消费者（技术分析/指标/回测/策略）均须使用会话感知K线

## 核心原则

子周期K线须按**交易所交易时段**划分，非简单自然钟均分分钟线。
"交易时段决定切片窗口，自然钟决定切点"。

## 各品种120m K线结构（每日bar数）

| 夜盘类型 | 代表品种 | 每日bar数 | bar时间戳 |
|:--------|:--------|:--------:|:----------|
| 23:00收盘 | SP/FG/RB/MA/TA/P/FU/SA/C/CF/SR等 | **3** | 23:00, 11:15, 15:00 |
| 01:00收盘 | CU/AL/ZN/PB/NI/SN/SS/BC等有色 | **4** | 23:00, 01:00, 11:15, 15:00 |
| 02:30收盘 | SC/AU/AG | **5** | 23:00, 01:00, 09:30, 13:45, 15:00 |
| 无夜盘 | SI/LH/JD/PK/AP/CJ/SF/SM等 | **2** | 11:15, 15:00 |

## 会话边界检测

**阈值**: 相邻bar时间差 > 120分钟 → 新会话

数学验证通过（2026-07-10）:
- 最大会话内间隔: 75min (10:00→11:15, 含10:15-10:30小节休)
- 最小跨会话间隔: 180min (11:15→14:15, 午休)
- 120min阈值安全居中: (75, 180) → ✓

## 关键注意点

1. **10:15-10:30小节休**: 商品60m bar体现为10:00→11:15的75min间隔, 不触发会话边界
2. **夜盘归属**: 周五夜盘→下周一K线 (日线级别影响, 子周期不受)
3. **4H周期**: 23:00收盘品种4H凑不齐(需21:00-01:00才够), 不推荐使用
4. **中金所9:30起**: 目前扫描品种不含中金所, 如需扩展到IF/IC/IM/IH需区分
