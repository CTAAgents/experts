# 定性信息门户目录（Qualitative Info Portals）

> **定位**：本目录是期货交易辩论专家团的**定性信息源总目录**，由团队主管（明鉴秋）在需补充
> **政策 / 研报 / 产业动态 / 现货基差 / 持仓排名** 等**无法由程序自动拉取**的定性证据时优先查阅。
>
> **与定量数据源隔离**：`data_sources.md` 中的 A/B/C/D 可靠性评分体系**仅评定量 feed**（价格 / 持仓 / 库存等
> 可由程序拉取的源）。本目录所列门户为**人工查阅的定性源**，不参与自动化置信度折扣
> （置信度统一按 1.0 处理，详见团队数据架构约定）。
>
> **分类框架**：按「权威层级 + 聚合度」分为三层。**第一层为监管/交易所（权威源头，免费）**，所有分析的基准；**第二层为综合资讯聚合站**，行情/快讯/研报/社区一体化；**第三层为产业垂直聚合**，按品种锚定现货+期货+基差。末附机构级付费终端参考。
>
> **来源**：金瑞期货微服务《期货信息去哪查？1 份权威网站清单》（2025-12-03）+ 资深交易员补充（2026-07-07）。

## 第三层：产业垂直聚合（按品种锚定现货+期货+基差）

做基本面/套保，光看综合站不够，需按品种锚定垂直站——它们把**现货报价+库存+开工+进出口+期货基差**聚合在一起。

### 黑色系
- **我的钢铁网** mysteel.com — 钢材/铁矿/煤焦全产业链现货+期货+基差
- **找钢网** zhaogang.com — 钢材现货成交、库存、物流
- **中国煤炭资源网** sxcoal.com — 动力煤/焦煤/焦炭产业链数据

### 有色系
- **上海有色网** smm.cn — 铜铝铅锌镍锡等现货报价+库存+升贴水
- **长江有色** ccmn.cn — 有色金属现货基准价
- **上海金属网** shmet.com — 金属市场综合报价

### 能化系
- **隆众资讯** oilchem.net — 石化/塑料/橡胶产业链开工+库存+价格
- **百川盈孚** baiinfo.com — 化工/新能源材料价格监测
- **卓创资讯** sci99.com — 能源/化工/塑料/钢铁等产业基准价与报告
- **金联创** 315i.com — 能化产业链资讯与定价

### 农产品
- **我的农产品网** myagric.com — 豆粕/玉米/生猪等现货+基差
- **美国农业部 USDA** www.usda.gov — 全球农产品供需平衡表
- **中国糖业协会** chinasugar.org.cn — 白糖产销数据

### 国际衍生品
- **CME Group 中文** www.cmegroup.com/cn — 国际衍生品行情与规则

### 通用基差工具
- **生意社** 100ppi.com — 500+大宗商品基准价，免费看基差
