# 信号因子研究笔记：通道突破与伪突破过滤在期货上的实证表现

> **来源**：2026-07-14 基于公开主流因子库与 CTA 实证文献的 Web 调研（WorldQuant BRAIN、QuantConnect 系、聚宽/海龟、breakoutos 2500+ 策略研究、NexusFi、ichihedge、国内期货圈共识）。
> **目的**：为 FDT 通道突破信号层（`channel_breakout_strategy.py` + `scan_all.py` P0-4 门禁）的因子筛选与伪突破过滤增强提供实证依据。
> **性质**：研究笔记（知识资产），非代码变更。

## 一、通道突破因子在期货（CTA/商品）上的表现

| 因子 | 期货实证数据 | 评级 |
|:-----|:------------|:-----|
| **Donchian DC20/55** | 中国 31 商品期货 2007–2022 回测：年化 20%、MDD 35%、Sharpe 0.41、胜率 33%（ichihedge）；纯突破在黄金已衰减到 PF≈1.06（fractiz GC），NQ 仍 PF=4.28 | ⭐⭐⭐ 基石但衰减 |
| **ATR/Keltner 通道** | 与 Donchian 同类，ATR 归一化对多品种横向比较友好（海龟用 N=ATR 做头寸） | ⭐⭐⭐ |
| **Bollinger 突破** | 均值回复/震荡品种（部分化工、农产品）更好；趋势品种不如 Donchian | ⭐⭐ |
| **动量突破** | 趋势确认用，单独做信号衰减快 | ⭐⭐ |
| **RSRS** | 聚宽股票特色，期货无实证 | ⭐ 不推荐 |
| **通道宽度/squeeze** | 蓄势识别，辅助维度 | ⭐⭐ |

**关键认知（NexusFi）**：Donchian 有"危机 alpha"属性——2008/2020/通胀冲击时多市场同时出现强方向性，正是它最强的环境；代价是震荡市连亏 15–20 次（每次约 1% 权益）。→ 突破+过滤在趋势性商品（黑色/能源/有色）适配好；低波动化工链需配均值回复逻辑。

## 二、伪突破过滤器在期货上的表现（按实证增益排序）

基于 breakoutos 对 **2500+ 指数期货突破策略**的研究 + 国内期货圈共识：

| 过滤器 | 期货实证增益 | 过拟合风险 | 评级 |
|:-------|:------------|:----------|:-----|
| **波动率择时（ATR bins）** | Nikkei 225 平均交易规模 **+91%**、回撤 **−79%** | 低（粗分箱） | 🔴 最强增益 |
| **成交量确认（>1.2–1.5×均量）** | 国内期货圈共识，过滤 70–85% 假突破 | 低 | 🔴 FDT 已有（vol_ratio 前置） |
| **ATR 幅度过滤（>0.5×ATR）** | 共识，过滤"毛刺突破" | 低 | 🔴 对应 P0-4 增强项 B |
| **时间确认（1–3 根维持）** | 共识 | 低 | 🟡 对应增强项 C |
| **close_position 实体质量** | 动能分用 close_position 0.6–0.95；长上影=假突破 | 低 | 🟡 对应增强项 A |
| **多周期共振** | 有效但实现复杂 | 中 | 🟡 对应增强项 E |
| **RSI 等阈值优化过滤器** | breakoutos 明确列为过拟合陷阱 | 高 | ⚪ 别碰 |

**核心原则（breakoutos）**：基于**结构市场条件**（趋势方向、波动率状态）的过滤器才稳健；基于指标阈值优化（如 RSI>63.5）的过滤器是过拟合——样本外必崩。过滤器应"零参数或粗分箱"以抗过拟合。

## 三、对 FDT 信号层的含义

1. **P0-4 门禁定位正确**：本质为结构型过滤（末根 vs 前 20 根极值 + spike 上限），非参数优化型，契合期货稳健原则。
2. **已覆盖**：成交量确认（vol_ratio 前置）+ spike 上限。
3. **缺口（按期货实证优先级重排）**：
   - 🔴 **B（ATR 幅度下限，突破幅度 < k×ATR 视为毛刺）** —— 升最前
   - 🟡 **A（close_position 实体质量，长上影=假突破）**
   - 🟡 **C（时间确认，日线扫盘下受限）**
   - 🟡 **E（多周期共振，有效但复杂，可后做）**
   - ⚪ **RSRS 划掉**（偏股票，期货无实证）
4. **落地铁律**：只做结构型 / 零参数过滤，拒绝指标阈值优化型。
