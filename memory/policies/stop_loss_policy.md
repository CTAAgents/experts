# 止损/目标参数 — 品种级与ADX动态调整

> 由闫判官在辩论方案中引用。参数通过历史数据验证。

## 品种级止损倍数（ATR乘数）

| 品种 | 止损倍数 | 说明 |
|:----:|:--------:|:-----|
| SA(纯碱) | 1.5×ATR | 默认值，已验证 |
| RB(螺纹钢) | 1.8×ATR | 触发率35.3%合理 |
| NI(沪镍) | 1.8×ATR | 误杀率3.8%可接受 |
| FU(燃油) | 2.0×ATR | ATR%=5.0%高波动品种 |
| 其他品种 | 1.5×ATR | 默认值，待验证后更新 |

## 品种级T2目标倍数（R乘数）

| 品种 | T2目标 | 达成率 | 说明 |
|:----:|:------:|:-----:|:-----|
| SA(纯碱) | 2.5R | 62.9% | 实际能走更远 |
| SM(锰硅) | 1.2R | 41.4% | 2R高估，已调低 |
| 其他品种 | 2.0R | — | 默认值 |

## ADX分级止损调整

| ADX区间 | 调整幅度 | 说明 |
|:-------:|:--------:|:-----|
| <25 | 默认 | 弱趋势 |
| 25-40 | 默认 | 中等趋势 |
| 40-60 | 缩小0.2-0.3x | 强趋势，缩小止损防被洗 |
| >60 | 缩小0.3-0.5x | 极强趋势，大幅缩小止损 |

## 参数更新规则
1. 运行验证脚本 → 更新此表 → 同步更新 debate-trading-planner SKILL.md
2. 每季度重新验证一次（样本量≥200行/品种时可信）
3. 标注参数版本（格式：v20260701）
